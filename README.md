# cd

## Manual

- `$ cd`: go to $HOME
- `$ cd /PATH`: go to absoulte path PATH
- `$ cd PATH`: go to relative path PATH
- `$ cd -1 PATH`: Save current path in register 1
- `$ cd -1`: Go to path stored in register 1
